Accessible Name and Description Computation 1.1

Accessible Name and Description Computation 1.1 is a set of rules that tells a web browser how to figure out the name of something on a webpage, like a button, link, or image. This name is what screen readers read out loud to blind users. It helps the computer know which text to use—like alt text, labels, or aria-label—so assistive technology can describe the item correctly.

It was created by the Accessible Rich Internet Applications (ARIA) Working Group at the W3C (World Wide Web Consortium)

The purpose of the Accessible Name and Description Computation 1.1 standard is to make websites easier for people using assistive technologies—like screen readers—to understand. It tells web browsers how to figure out the correct name or description for things on a webpage, such as buttons, images, links, and form fields.
